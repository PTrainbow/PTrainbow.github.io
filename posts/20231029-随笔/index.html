北邙山之光
 2023-10-29  约 1307 字   预计阅读 3 分钟 
目录

因为工作和生活的事情，最近让自己变的很抑郁，甚至每天下班必须要散步，晚上一定要听歌，不然就会胡思乱想，觉得人生没有意义。有时候觉得自己每天耗费时间去学这学那，什么 JVM 啊，什么 AGP 啊，意义又何在呢？

其实，我最近十分想逃离现在的工作环境，所以也尝试去面试过两家公司。

一家聊得好好的，3面都结束了，以为没问题了，结果莫名挂了···另一家一面的问题就莫名其妙，面完就 GG 了···然后周六又被捞起来，也是一堆神奇的问题，虽然是 Android 基础技术岗位，但是似乎是想做底层的通用架构层，所以问我会不会写 C++，甚至聊了一些 C++ 相关的面试题，这可是难为住我了，我平时自己写写还可以，但是 C++ 的一些面试题和一些高级特性，那还真是答不上来，顺便还被吐槽了跳槽频繁😁，估计也是挂了吧(虽然我倒是也挺想转行做底层开发，奈何没机会)~~

其实放眼看去，互联网公司全都在走下坡路吧。老板们其实都已经是盆满钵满了，只不过还得装装样子，要追求“收益”。那下面的人总有还没盆满钵满的，那就会促使他和他的下属卷起来~~

很多人也告诉我，我司已经是为数不多还在招人的公司了，不如继续苟活，降低预期即可

但是，看着一堆堆“技术专家”那种囧态，那种很虚很假的样子，而自己的绩效却被人家握在手里不得不低头跪舔~~这一切的一切都让我更加的烦躁和抑郁，逐渐崩溃

想起了曾经中考的黑暗时刻，因为没考好，每天都被父母高强度摧残内心，那个假期没有吃过一顿好饭，每天都要流眼泪，每天都要窝在床边被训斥。但是那个时候我打篮球、看 NBA，我喜欢科比，我的注意力会转移，也有一些精神上的寄托。现在科比也不在了，篮球也不打了，体型逐渐圆润，也变成肥宅了~~唯一让我重燃热血的就是 DOTA 了。这一个月，我反复靠着 DOTA 和 TI 给自己打鸡血，让自己快乐了起来。但是今天早上看完胜者组决赛后，十分的绝望，道心破碎···我可能比场上的选手还要难过，我觉得我一切的快乐都没了···

看完比赛以后，我从 12点 开始睡觉，迷迷糊糊，醒来五点多了，但是明显感觉还是很难受···

什么也不想做，就漫无目的的翻着抖音。曾几何时，我每天都说它是精神鸦片。可是就在今天，我发现我被推送了一个 《我被美女包围了》 这样一个游戏视频，这个游戏大致就是你可以按照自己的选择和各个女生交往，以达成各种结局。本来是一个很垃圾的游戏，但是当今的中国社会，有无数缺爱的游戏屌丝，我也是。于是，它冲上了 steam 榜单。并且，我看到了一个巨搞笑的视频，抖音链接如下：

6.17 D@h.BT 05/14 dAG:/ 复制打开抖音，看看【生活随拍 可能有关】# 精神状态belike # 纯爱 # 郑梓妍 所以什么是爱情呢？# 精神状态belike # 纯爱… https://v.douyin.com/idcs1otk/

我感觉我一下被治愈了···

一方面是这个游戏和视频实在是太搞笑了😄

另一方面是虽然缺爱的人如此之多、当今的环境如此艰难、我们都知道这是一个搞乐子的游戏，但是大家还是毅然决然的花了30多块钱买下了这个游戏。大家都是抱有希望的、都是乐观的、都还是有纯真的，所以又让我对人性本善有了一丝丝的相信，又让我振作了起来

总之，目前啥都不事儿吧。生命本身就是一种体验，如果当真思考了个明白，觉得活着没意义，也不用急着寻死

先出去走走，也许碰到了自己爱的地方，爱的人，又觉得活着有意思了呢？
