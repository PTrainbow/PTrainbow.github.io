又是一年 6.30，毕业五年了！

自己也变成了“老白兔” “前辈” 了，也越来越不会让工作占据生活。

更佛、更淡定、也更坚定，也掌握了一种躺平的哲学。

现在，居然也有人问我类似 “选大厂还是小厂” 这种问题了。

感觉这种问题确实没法回答~

问题不大，围绕着几个我最近在思考的问题，写下一篇 “哲学” 总结

有关应届生选大厂还是小厂

互联网大小厂的，我也都体验了一下，以我的跳槽频率，到也算是有一点点资格来回答这个问题。

如果你只是为了赚钱，我强烈建议大厂混三五年跳去小厂，作威作福(是真的可以作威作福哦)。

如果，你对写代码有点点热爱，我可能会说：看你的 leader，只要 leader 足够的优秀，大厂小厂没有什么区别。大厂也有一帮废物，小厂也藏有大神。可是和你息息相关的，只有你的 leader。

不过，对于校招的话，你可能都不知道自己在哪个组···所以，看命吧。

有关命和觉醒

苟全自己

我最近属于一个三观崩塌，然后重建的过程。

为什么崩塌呢？主要是我经历了很多奇奇怪怪的事情，无论是工作还是生活，我都有很多神奇的经历。

认真的说，有时候，也挺想去修道的(现实主义者轻喷)。一般这么说的人，往往是一个“失败者”的身份，或者是逃避现实啊之类的。

我不知道我算不算，可能也算？我只是觉得一切本应该很简单，为什么现在要搞得这么累。

听到了好多有关谈婚论嫁的事故，没错，是事故。感情这种东西，自古以来就是不干不净，现在更加的不干不净了。不过，现在也有一小撮人，在渐渐的“去封建”、“去奴役”，这东西是影响很大，但是主要的两个人觉得没问题，你们瞎操什么心呢？一个人能活多久呢？那为什么要这么多的“脸面”呢？

这只是其中一个吐槽较多的点，还有许多事情，我都觉得挺麻烦的。

另一方面，以前源儿哥说我觉醒的太早了，我觉得是我觉醒的太晚了。

我一直想不明白为啥大家要这样子活着，我感觉人均“恶人”。

不过，我现在想明白了。大家都是为了自保而已，大家都是抱着对别人“恶意”或者说最自保的心态做人了，只有我还是像个愤青。当我把我放入到一个极尽全力苟全自己的环境中去，我一下子就理解了很多我觉得神奇、甚至充满诡异的事。

而愤青们往往是带有理想主义色彩的，更像是小说中的“大侠”，甚至可以为兄弟抛头颅洒热血。可惜，大部分的“兄弟”都是已经觉醒的现代人，“兄弟”只会看着你的热血，偷偷的笑，这人是不是有点呆、有点不成熟。

嗯，各有利弊吧。

信命

再来说说，命。

有的时候，你不得不信命，

前段时间我还看了自己以前的朋友圈，那时候大三在实习，我觉得那次实习一直是人生的伏笔，一个大公司莫名的招了我这个菜鸡，也莫名的让我从一个 java 后端成为了 Android，又莫名的没人管，让我觉得自己菜到有点丢人，自己离职，又不服气，又学起来了 Android，是不是很搞笑。

我当时要是这些年的一些思考，我可能不会去学 Android 了，说明当时的命认为我学 Android 并不好，在默默的阻挠我，并且一直在为我选一条更好的路。

现在回想起来，这个阻挠一直在持续，甚至到今天。所以我也偶尔看一些其他的东西，只不过仅仅是看，没有动手，也觉得挺有意思的，各个新东西、老东西 hello world 一下，像是看小说一样。

这里，可能有点玄学了。我也一直在研究玄学，玄学贯穿了我这工作的几年，说出来，全当一乐。

从我刚毕业去了京东，手里就一直有非 Android 相关的事儿，当时会写一些 python 脚本。后来去了滴滴，写上了 react/vue 和 go。可能也就在快手，手里的事都是 Android，但是主要做的是 TV，搞的是焦点和遥控器。后来去小度，留在那里的话，后面也要从事后端开发，组里都是后端，特别对不起小度，当时想学学 databinding 就直接开搞了，留下了一坨屎。甚至如今在小红书，也写上了 python。

你能说，这不是命吗？

所以，我早就和解了，写什么并不重要，可能命运认为我不应该把重心放在 Android 上？三番五次的打岔。

而且，最近就突然越来越想重新回去读书，找到一个自己喜欢的技术领域，比如编译器、图形学、AI这种，无所谓前端后端客户端。所以，感觉又被指引了，可是目前还需要一些米，所以还得再等等。

不过，可能过段时间，命运就变了，指引我去送外卖、跑滴滴了。

有关技术

个人提升的话，可能内功有了一点点的精进吧，外功感觉停滞不前了。

对于技术的看法也有了一些改变

技术到底重不重要？不知道咋回答了···

可能，在互联网公司混，表达能力 和 ppt能力 更重要吧。

比如，和一些“资深研发”聊技术需求，聊着聊着就跑偏了，直接被反问了~~技术问题丝毫没有解决，反而牵扯进来了各种业务问题

今天还有个好玩的现象，某厂商的研发直接被问的我感觉他是有点无语住了，可能从未见识过互联网的卷法吧，哈哈哈哈！

技术方向的话，c/c++ 一直是我的弱项，从没有写过一个 c/c++ 商用项目。涉及到一些 native 库，也只是修修补补，改改编译。可是，反过来这个东西目前的应用场景和我的工作场景完全不搭的，我也很难接触的到，而且古老语言的开发效率也比较低，所以我又想看看 rust。不过，可能又是三分钟热度吧。

操作系统的话，xv6 搞下来了，然后做了一个 lab，就闲置了，没太多提升。

我越来越感觉，其实技术的理论模型是很重要的。

比如，我看 ART 的源码，经常找不着北，我感觉有时候无法理解他们的设计模型。再比如，我看 R8 的源码，也是摸不着北，也是因为我对于这种字节码优化的理论模型一点都不清晰。

感觉这些东西，都是一块专门的领域，需要时间和基础知识慢慢来，可能留学读书学习是一个更好的选择。

有关工作和生活

不在遥远的五环了，可是生活并没有发生什么变化。

在北京的朋友更少了，海底捞也找不到人吃了，更别说其他的了。

周末，因为自己的 windows 本跑不动，也很少玩端游了，更多的是刷剧和手游和听歌。

喜欢摸一摸纸质的书，但是看两页就困。

更热爱研究玄学，也喜欢算卦，信则有不信则无。

工作和生活分的特别开，周末坚决不看工作，除非你告诉我小红书因为我的代码炸了，那我也能火一把。

思想更加的简单和佛。一件事情不开心，我就不会去做。有不开心的人，我就尽量和他没有交集。

不过，依然有时候做了一些事，又觉得自己太冲动，后来想想，其实忍一忍也就过去了。

总结

生活中的各种经历和见闻，也都让我更深刻的认识了，什么是无我和无常。

咱就是说，佛、道、玄 三修吧！

工作不是人生，好好生活。

贫困潦倒，不影响精神富足。
